# concesionario
## Como Poner en marcha el proyecto.
1. Tener instalado el JDK25 y el dbeaver con mariadb vinculado
2. Tener descargado el archivo zip con el proyecto
3. Cargar el Proyecto en IntelliJ

## Introducción y propósito de la aplicación.
La aplicación tiene como objetivo crear un sistema sencillo para gestionar un concesionario de vehículos, permitiendo administrar coches, clientes y ventas de forma coherente. Simula el funcionamiento real de un concesionario y pone énfasis no solo en la funcionalidad, sino también en la correcta estructuración del código y el uso de una arquitectura clara y profesional.

## Descripción general del funcionamiento.
La aplicación dispone de una interfaz gráfica en JavaFX que permite gestionar visualmente coches, clientes y ventas. La información se almacena de forma persistente en una base de datos MariaDB y se actualiza con cada acción del usuario. El sistema respeta las relaciones del modelo entidad-relación, garantizando la coherencia entre clientes, ventas y coches.

## Tecnologías utilizadas.
Para el desarrollo de la aplicación hemos utilizado las siguientes tecnologías:

● Java como lenguaje principal de programación.

● JavaFX para la creación de la interfaz gráfica de usuario.
● MariaDB como sistema gestor de bases de datos relacional.
● JDBC para la conexión entre la aplicación Java y la base de datos.
● Maven para la gestión de dependencias y la estructura del proyecto.
● IntelliJ IDEA como entorno de desarrollo.
● Git y GitHub para el control de versiones del proyecto.
